# react-shopping-cart

## About
This repository is my practice project about simple shopping cart.

## Features
* Login and logout
* The prouduct add to the shopping cart
* Remove product in cart.
* Edit member profile.
* Update password

## Used technologies
* React
* redux
* react-router
* react-router-dom
* react-bootstrap
* react-icons
* firebase
* styled-components

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn start
```

### Compiles and minifies for production
```
yarn build
```
